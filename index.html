JS TLC: Web Optimization

Prerequisites

  • Node.js (v0.12.7 or v4.x.x preferred)

    Download: https://nodejs.org/

    nvm-windows for managing multiple versions of Node

  • Bower

    http://bower.io/

  • Grunt

    http://gruntjs.com/

  • Git

    Git for Windows

    Sourcetree (GUI client)

  • GitHub account

    http://github.com/

  • ngrok

    https://ngrok.com/

Activity

  1. Fork the repo: github.com/arkon/jstlc
  2. Clone your forked version to your machine.
  3. Follow the README instructions to set it up.
  4. Fix it up!
  5. Benchmark it (see below).

Benchmarking

  • YSlow

    http://yslow.org/

    YSlow runs locally in your browser.

  • WebPagetest

    http://www.webpagetest.org/

    WebPagetest requires an actual URL, so you can use ngrok to expose your local server.
