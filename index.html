JS TLC: Web Optimization

Prerequisites

  • Node.js (v0.12.7 or v4.x.x preferred)

    Download: https://nodejs.org/

    nvm-windows for managing multiple versions of Node

  • Bower

    http://bower.io/

    $ npm install -g bower

  • Grunt

    http://gruntjs.com/

    $ npm install -g grunt-cli

  • Git

    Git for Windows or Sourcetree (GUI client)

  • GitHub account

    http://github.com/

  • ngrok

    https://ngrok.com/

Activity

  1. Fork the repository: github.com/arkon/jstlc
  2. Clone your forked version to your machine.
  3. Follow the repository's README instructions to set it up.
  4. Fix it up!
  5. Benchmark it (see below).
  6. Enter your scores in this form.

Benchmarking tools

  • YSlow

    http://yslow.org/

    YSlow is a local browser extension. Simple install it and use it from your browser's toolbar.

  • WebPagetest

    http://www.webpagetest.org/

    WebPagetest requires an actual URL, so you can use ngrok to expose your local server. From a terminal in the directory where you placed ngrok, run:

    $ ngrok http 8080
